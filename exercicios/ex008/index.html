Principais formatações

Negrito / Destaque

Existem duas formas de colocar um texto em negrito: na forma semântica e não semântica.
Veja um exemplo de frase na forma não semântica, isto é, sem significado:
Essa frase está em negrito. Foi criada pelo código "b" no próprio HTML5.
Não possui significado (semântica), apenas num código ultrapassado de estilo.

Agora, vamos utilizar um código que também deixa o texto em negrito, entretanto, sua finalidade não é puramente estética.
Utilizando o código "STRONG", é possível deixar uma frase em negrito, mas a sua finalidade é DESTACAR algo importante, ou seja, dar ÊNFASE, e não apenas formatar.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag "I" (não semântica).
Já nessa frase, aparentemente também temos um termo em itálico, PORÉM, foi escrito através do código "em", que significa "ênfase".
Claro, o seu sentido também é semântico, pois possui um significado, que é dar ênfase a algo.

Observações:

A pesar de ambas as formas (semântica e não semântica) serem válidas atualmente,
Gustavo Guanabara recomenda que só use as semânticas, pois muitas tags antigas do HTML podem deixar de funcionar (como as tags "b" para negrito e "i" para itálico).
Então lembre-se: "strong" para destacar e "em" para dar ênfase.

Texto marcado

Podemos também criar um texto marcado usando a tag "MARK".

Há duas maneiras de "colorir" a marcação do texto.
A primeira é chamando o CSS dentro da marcação "mark" ("mark style=background-color: lightpink;">
E a segunda é indo lá no cabeçalho (head) e chamando o CSS, que é o que vou fazer agora.
O legal de usar o CSS no cabeçalho é que a tag "mark" funcionará em todas as marcações.
Se você quiser uma marcação específica, poderá colocar o próprio CSS ao lado da tag.

Texto sobrescrito

Para inserir coisas do tipo x20+3. Basicamente, a tag "sup" serve para elevar as coisas ao expoente, ex: x2 ou n15.

Texto subscrito

Para inserir coisas do tipo H2O. Já a tag "sub" serve para abaixar as coisas, neste caso, o número 2 de H2O.
