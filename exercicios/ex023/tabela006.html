Tabelas em telas pequenas

Quando a tabela é muito extensa, acaba gerando um problema de rolagem. Para evitar estresse, principalmente para quem usa aparelhos móveis, vamos envelopar a tabela dentro de uma div. Essa div vai nos permitir aplicar algumas propriedades para melhorar o uso da tabela.

Agora que colocamos a tabela dentro da div e aplicamos um id específico para ela, vamos configurar para deixar a nossa tabela mais flexível e rolável. No CSS, escreva overflow: auto. Isso já será o suficiente para deixar a tabela rolável sem alterar o conteúdo do site. Overflow significa algo que flutua, que transborda, que sai de parâmetro. Deixando-a em automático, o navegador se encaminhará de aplicar a rolagem da tabela.
