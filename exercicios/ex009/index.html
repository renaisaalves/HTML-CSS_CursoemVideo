Trabalhando com Listas

Listas Ordenadas

Na lista ordenada, existem 5 tipos que podem ser ordenada. A mais simples é a do tipo numeral,
representada pelo código <ol type="1"> <li> alguma coisa </ol>

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Do tipo alfabético MAIÚSCULO <ol type="A"> <li> alguma coisa </ol>

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Do tipo alfabético MINÚSCULO <ol type="a"> <li> alguma coisa </ol>

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Do tipo romano MAIÚSCULO <ol type="I"> <li> alguma coisa </ol>

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Do tipo romano MINÚSCULO <ol type="i"> <li> alguma coisa </ol>

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Além disso, se você quiser inserir uma outra numeração, letra ou númer romano, que não comece com 1, o código é:
<ol type="escolha a letra" start="escolha a ordem que você quer começar, exemplo o 5"> <li> alguma coisa </ol>

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Listas Não Ordenadas

Para Listas Não Ordenadas, existem quatro tipos: disc, circle e square.
Vejamos o exemplo em disc: <ol type="disc"> <li> alguma coisa </ol>

  • Pão
  • Leite
  • Pão de queijo

Em formato de circle: <ol type="circle"> <li> alguma coisa </ol>

  • Pão
  • Leite
  • Pão de queijo

Em formato de circle: <ol type="square"> <li> alguma coisa </ol>

  • Pão
  • Leite
  • Pão de queijo

Lista de Definição

Os códigos <dt>Termo</dt> e <dd>Descrição</dd> atribuem um nome (termo) e sua descrição.

HTML
Descrição = Linguagem de marcação para a criação do conteúdo de um site.
CSS
Descrição = Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
