Código-fonte / Pré-formatação

O comando document.getElementaryById('teste') é escrito em linguagem JavaScript

Observe que o comando está escrito em uma fonte que facilita a compreensão da leitura.
Se o comando estiver escrito em uma fonte comum, a visualização do código seria dificultada, visto que cada letra possui um espaço diferente.
Então, para facilitar a visualização do código, os programadores optam pela tag "code" que espaçam as letras.

Além disso, há uma outra questão muito importante a ser dita.
Quando copiamos e colamos um código, ele fica literalmente reto, sem identação. Observe o exemplo abaixo:

num = int(input('Digite um número')) if num % 2 == 0; print (f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

Viu como ficou ruim? É por isso que a identação é muito importante, para manter o código organizado, da maneira como ele foi escrito.
Veja um exemplo abaixo, de um código escrito em Python, agora com identação:

    
        num = int(input('Digite um número'))
        if num % 2 == 0;
        print (f'O número {num} é PAR')
        else:
        print(f'O número {num} é ÍMPAR')
        print('Fim do programa')

Para deixar um código com identação, basta colocar o código dentro da tag "PRE". A lógica seria: "CODE" dentro de "PRE".
Um outro macete é apertar SHIFT + TAB para fazer o código se deslocar para a esquerda. Dessa forma:

        
num = int(input('Digite um número'))
if num % 2 == 0;
print (f'O número {num} é PAR')
else:
print(f'O número {num} é ÍMPAR')
print('Fim do programa')

Você também pode apertar somente a tecla TAB pra fazer com que o código se desloque para à direita.
A deslocação do código se chama TABULAÇÃO.
